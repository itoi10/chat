# [Chatbot](https://chatbot-a99d5.web.app/)

## 概要

YouTube 講座を参考に React でチャットボット作成

参考元は JavaScript だが TypeScript に置き換えて開発

## 導入方法

```
yarn install
yarn start
```

コードフォーマット

```
yarn lint
```
