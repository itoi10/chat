# Chatbot

## Description

YouTube 講座を参考に React でチャットボット作成

参考元は JavaScript だが TypeScript に置き換えて開発

## Deploy

[Firebase](https://chatbot-a99d5.web.app/)

## Setup

```
yarn install
yarn start
```

Run eslint and prettier

```
yarn lint-fix
```
